# UED landing

Please use pnpm.


## Get Start

- 1、static page

``` bash
pnpm run dev
```

- 2、dev node server

``` bash
pnpm run dev:node
```

- 3、lint

``` bash
pnpm run lint
```


## Deploy

``` bash
pnpm run deploy
```
